# 🦞 Lobster Platformer

用 Godot 4 制作的 2D 平台跳跃游戏！

## 🎮 游戏内容

- ✅ 玩家角色（小龙虾 🦞）
- ✅ 平台跳跃
- ✅ 收集金币（得分）
- ✅ 星星收集（每个 50 分）
- ✅ 掉落会死亡
- ✅ 敌人巡逻 + 飞行敌人
- ⚔️ **Boss 战** - 挑战 Red Dragon！
- 🏆 **胜利画面** - 通关后显示
- ⭐ **Power-up 系统** - 无敌、速度、二段跳、额外生命
- ⏱️ **计时挑战** - 记录通关时间
- 🏆 **成就系统** - 8个可解锁成就

## 🎯 游戏特色

- **12个精心设计的关卡**
- **连击系统** - 快速收集金币获得分数加成
- **Boss 战** - 击败红龙获得丰厚奖励
- **计时挑战** - 更快通关获得时间奖励
- **成就系统** - 解锁各种成就
- **高清分数记录**

## 如何运行

### 1. 下载 Godot
访问 https://godotengine.org/download 选择 **Godot 4.2.x** 的 **Standard** 版本（不是 .NET）

### 2. 打开项目
- 解压下载的 Godot
- 运行 Godot
- 点击 "Import" 
- 选择这个文件夹的 `project.godot` 文件
- 点击 "Import & Edit"

### 3. 运行游戏
按 **F5** 或点击播放按钮

## 操作说明

| 按键 | 功能 |
|------|------|
| ← / A | 向左移动 |
| → / D | 向右移动 |
| 空格 / ↑ / W | 跳跃 |
