# 🦞 Lobster Platformer

用 Godot 4 制作的 2D 平台跳跃游戏！

## 如何运行

### 1. 下载 Godot
访问 https://godotengine.org/download 选择 **Godot 4.2.x** 的 **Standard** 版本（不是 .NET）

### 2. 打开项目
- 解压下载的 Godot
- 运行 Godot
- 点击 "Import" 
- 选择这个文件夹的 `project.godot` 文件
- 点击 "Import & Edit"

### 3. 运行游戏
按 **F5** 或点击播放按钮

## 操作说明

| 按键 | 功能 |
|------|------|
| ← / A | 向左移动 |
| → / D | 向右移动 |
| 空格 / ↑ / W | 跳跃 |

## 游戏内容

- ✅ 玩家角色（小龙虾 🦞）
- ✅ 平台跳跃
- ✅ 收集金币（得分）
- ✅ 掉落会死亡
- ✅ 敌人巡逻
